Microsoft Passwordless Account

Ich habe irgendwann mal zwei "Yubikeys" (Google Titan Key) gekauft und mit meinem Microsoft-Account "verknüpft". Microsoft hat mir sogar angeboten meinen Account auf "passwordless" umzustellen. Das erforderte aber, dass man die Microsoft Authenticator App als zweiten Faktor einrichtet.

Ja, gut, habe ich gemacht. Danach konnte ich das Passwort meines Microsoft Kontos sogar entfernen. Coole Sache, nich nich?

Heute kam ich dann auf die glorreiche Idee in meinem Microsoft-Account einen zusätzlichen OTP Code hinzuzufügen. Das bot mir die Webanwendung von Microsoft an und ich habe den OTP Code ohne Probleme hinzufügen können. Der Hintergrund: ich wollte den OTP-Code in Apple "Passwörter" haben, so dass ich am PC nicht die Android-App starten muss, um mich bei Microsoft einzuloggen. Ich wollte zusätzlich die Microsoft App auf meinem Smartphone los werden.

Ich habe danach versucht "Microsoft Authenticator" auf der Profilseite als zweiten Faktor zu entfernen. Brauche ich ja nicht mehr, dachte ich! Microsoft hat mir aber einen Dialog angezeigt, dass dieser zweite Faktor nicht entfernt werden kann. Ok. Dann halt nicht.

Ich melde mich ab, um zu testen, ob ich mich mit dem OTP-Code einloggen kann. Bei dem Login sagt Microsoft mir aber, dass mein Account kein Passwort hat!? Ja, das habe ich ja letztes Jahr irgendwann so eingerichtet. Bis hier kein Widerspruch.

Und nun? Ich soll für meinen Account ein Passwort definieren. Weil "zweiter Faktor" aktiviert ist, fragt es mich nach einem OTP-Code, den ich eingeben kann. Den Code nimmt er auch. Aber jetzt will Microsoft mir einen Code per E-Mail zuschicken. Der Button funktioniert aber nicht. Ich kann ihn klicken, aber es passiert nichts.

Alternativ kann ich mir einen Code per SMS zuschicken lassen, aber es wird mir angezeigt, dass diese Überprüfungsmethode nicht zur Verfügung steht.

Joa, ich komm nicht rein. Ich habe zwar zwei Titan Keys hinterlegt, aber die Option dafür wird mir gar nicht erst angezeigt :D. Aus irgendeinem Grund ist mein Account nicht mehr "passwordless", aber immer noch ohne Passwort, weswegen ich nicht "rein komme". Ich vermute, dass mein Klick auf "entfernen" bei Microsoft Authenticator in den Profileinstellungen irgendwas ausgelöst hat.

Oder wie mein Kollege immer sagte: alles Schrott.

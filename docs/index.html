Projekte: Software & so
  1. tomcat-environment-property-source-file – Ein Tomcat Plugin, um Passwörter in den XML-Konfigurationen nutzen zu können, ohne, dass man Passwörter direkt in den Umgebungsvariablen stehen hat, "weil wegen" Sicherheit :)
  2. TILStory – Meine eigene Blogsoftware für Java / Jakarta EE Umgebungen, JPA, JSP, JSTL, CDI, Postgres, Flyway ... die Dockerumgebung nutzt Wildfly, wofür auch die Authentifizierung und Authorisierung konfiguriert ist
  3. tomcat-docker-image – Tomcat Docker Image mit vorkonfiguriertem Datenbankpool und anderen, für mich sinnvollen, Anpassungen
  4. LFS – meine Linux-From-Scratch Skripte
  5. flywaydbpostgres – Docker Image für Flywaydb, das nur Postgres unterstützt
  6. Stromzettel – Eine kleine Swing-Anwendung zur Erfassung von Stromzählerständen mehrerer Zähler
  7. FormDialog – eine Java Swing Bibliothek, mit der ich in einer Swing-Anwendung ohne viel Aufwand ein Formular erstelle
  8. DateTimePanel – Eine Java Swing-Komponente, mit der ich in meinen Swing-Anwendungen Datums- und Zeiteingaben erfasse und anzeige
  9. SThemes – Java Swing Steel Themes für das Metal Look & Feel
  10. SvgIcon – Eine Java Swing Komponente, mit der man SVG-Icons anzeigen kann
  11. LAFManager – Java Swing Look & Feel Manager
  12. AHrefLabel – Java Swing Komponente, mit der man ein JLabel in einen klickbaren Link verwandeln kann
  13. AboutDialog – Ein AboutDialog für Swing-Anwendungen.
  14. Settings
    1. Settings API – API für Settings
    2. Settings JSON – JSON Settings
    3. Settings YAML – YAML Settings
    4. Settings TOML – YAML Settings
  15. DAdmin – DAdmin ist eine Webanwendung zur Verwaltung von registrierten Domainnamen für Kunden inkl. Preisgruppen, TLD-Verwaltung usw...Perl 5 und CGI. Funktioniert heute noch ohne Anpassung :D! Die Anwendung ist von 2009.
